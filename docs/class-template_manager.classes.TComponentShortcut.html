Class TComponentShortcut

From instance of this class you can esier get to the methods of components (compontents of current template)

For example to call function getColorSheme() of TColorScheme component: $tcs = new TComponentShortcut; $colorScheme = $tcs->TColorScheme->getColorSheme(); or even shorter (upper case letter from name of component): $tcs->$TCS->getColorScheme();

(monostate)
